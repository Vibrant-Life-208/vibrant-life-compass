# Vibrant Life Family Playbook 2026-2027 - Canva Build Spec v1

> How to assemble the editable master in Canva and export the print-ready PDF with clickable video buttons.
> **Content source:** `2026-06-29-family-playbook-2026-2027-v0.6.md` (the words). **Visual target:** `Vibrant-Life-Family-Playbook-2026-2027-PREVIEW.pdf` (the look). **Videos:** `...-video-scripts-v0.2-additive.md`.

---

## 0. Workflow (the whole picture)
1. Build the playbook as a **Canva Doc or Presentation/Flyer (Letter, portrait)** - this is the **editable master** Jenna keeps.
2. Add the four **video-stop buttons** as clickable links pointing to the videos on **YouTube** (hosting decision, 2026-06-30).
3. **Share → Download → PDF Print** to publish. Links stay clickable in the PDF.
4. To update later: edit the Canva master, re-download. Swap a video by changing the button's link - no re-export of layout needed.

*Why Canva: it's the editable master Jenna maintains, it supports clickable link buttons, and it can also produce the videos (cartoon elements + AI voice + captions) - one home for the PDF and the films.*

## 2. Reusable components (build once, copy everywhere)

- **Section card:** white rounded rectangle (corner ~18px), 2px `#EADFC7` outline, soft shadow. Heading row = a small colored circle icon + the section title.
- **Reflection callout (the warmth):** soft green box `#F3F8EE`, a 4px left bar in `#7CC36A`, *italic* text, label **"Reflect with us:"** in `#4E9D54`. Use the four reflection questions from v0.6.
- **Video-stop button:** coral pill/box `#FF7A59` with a white play-triangle, bold title "Watch: [name]", small "(length)" + one-line description. **This is the clickable element** (see §4).
- **Studio card:** rounded rectangle filled in the studio color, white text (dark on Spark), with an age pill top-right.
- **Quote line:** small italic grey `#8A8068`, attributed.

## 4. The four video-stop buttons (clickable)

Place each as a coral button where the page map shows it:

| # | Button | Page | Links to |
|---|---|---|---|
| 1 | **Watch: Why Vibrant Life** (60-90s) | Welcome | `[video 1 URL]` |
| 2 | **Watch: A Day at Vibrant Life** (60-90s) | Daily Life | `[video 2 URL]` |
| 3 | **Watch: Studio Shorts** (30-45s each) | Fields of Play | `[playlist URL]` (or one link per studio short) |
| 4 | **Watch: What We Ask / Partner, Don't Rescue** (60s) | Parent Journey | `[video 4 URL]` |

**Hosting = YouTube.** Upload each video to the Vibrant Life YouTube channel. Privacy choice per video:
- **Unlisted** (not searchable; only people with the link see it) = the privacy-safe default - good if a video will ever include real guide/learner footage.
- **Public** = fine for the cartoon/TTS versions (no real children), and it helps prospective families find you. A reasonable split: Welcome public, the rest unlisted - your call.
- For the **Studio Shorts**, either make one **playlist** (link the button to the playlist) or link each short individually.

**To make a button clickable in Canva:** select the button group → click the **link icon** (or Ctrl/Cmd+K) → paste the YouTube watch URL. On PDF export, that becomes a live hyperlink. (Do **not** embed the video file in the PDF - it won't play on most devices; link out to YouTube instead.)
